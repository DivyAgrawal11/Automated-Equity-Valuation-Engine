# Automated Equity Valuation Engine

## Project Overview
An end-to-end Python pipeline designed to automate the equity valuation process for Indian markets (NSE/BSE). The engine ingests raw fundamental data, calculates trailing metrics, projects future cash flows, and executes a Discounted Cash Flow (DCF) analysis. The final output is an automated, institutional-grade PDF research report featuring sensitivity heatmaps, valuation waterfall bridges, and scenario analyses.

## Core Features
* **Automated Data Extraction:** Pulls live pricing and fundamental data via the `yfinance` API, converting metrics to INR Crores for standard Indian institutional reporting.
* **Dynamic Financial Modeling:** Calculates Cost of Equity (CAPM) and Weighted Average Cost of Capital (WACC) using local macroeconomic variables (e.g., 10-Year G-Sec yields, Indian ERP).
* **Logic-Gated Terminal Value:** Utilizes the Gordon Growth Model with programmatic safety checks to prevent mathematical errors when growth rates approach or exceed the cost of capital, gracefully handling `NaN` edge cases.
* **Custom Assumption Engine:** Allows for stock-specific inputs regarding base revenue growth, operating margins, and terminal growth rates to compute Base, Bear, and Bull valuation scenarios.
* **Automated Reporting:** Renders comprehensive two-page HTML-to-PDF reports using `Jinja2` and `pdfkit`, featuring custom `Plotly` visualizations.

## Tech Stack
* **Language:** Python 3.x
* **Core Libraries:** `pandas`, `numpy`, `yfinance`
* **Visualization & Reporting:** `plotly`, `kaleido`, `jinja2`, `pdfkit`
* **Environment:** Google Colab / Jupyter Notebook

## Installation

Because the engine relies on `pdfkit` to convert HTML templates into PDF documents, the system-level `wkhtmltopdf` binary must be installed alongside the Python dependencies.

```bash
# System dependencies (Linux/Debian environments)
sudo apt-get update -qq
sudo apt-get install -y wkhtmltopdf

# Python packages
pip install yfinance plotly pdfkit jinja2 kaleido==0.2.1
```

## Quick Start

Execute the valuation pipeline by defining your core thesis assumptions and passing the ticker symbol.

```python
from valuation_engine import run_v2_pipeline

# Define fundamental assumptions for the specific asset
assumptions = {
    'base_rev_growth': 0.08,     # 8.0% revenue growth
    'base_ebit_margin': 0.34,    # 34.0% operating margin
    'terminal_growth': 0.045     # 4.5% terminal growth rate
}

# Execute pipeline for the target equity (e.g., ITC Limited on the NSE)
run_v2_pipeline("ITC.NS", assumptions)
```
