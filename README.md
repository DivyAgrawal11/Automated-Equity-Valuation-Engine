# Automated Equity Valuation Engine

## Project Overview
An end-to-end Python pipeline designed to automate the equity valuation process for Indian markets (NSE/BSE). The engine ingests raw fundamental data, calculates trailing metrics, projects future cash flows, and executes a Discounted Cash Flow (DCF) analysis. The final output is an automated, institutional-grade PDF research report featuring sensitivity heatmaps and valuation ranges.

## Core Features
* **Automated Data Extraction:** Pulls live pricing and fundamental data via the `yfinance` API, converting metrics to INR Crores for standard Indian institutional reporting.
* **Dynamic Financial Modeling:** Calculates Cost of Equity (CAPM) and WACC using local macroeconomic variables (e.g., 10-Year G-Sec yields, Indian ERP).
* **Logic-Gated Terminal Value:** Utilizes the Gordon Growth Model with programmatic safety checks to prevent mathematical errors when growth rates exceed the cost of capital.
* **Custom Assumption Engine:** Allows for stock-specific inputs regarding base revenue growth, operating margins, and terminal growth rates.
* **Automated Reporting:** Renders HTML-to-PDF reports using `Jinja2` and `pdfkit`, featuring custom `Plotly` visualizations (Sensitivity Heatmaps and Football Fields).

## Tech Stack
* **Language:** Python
* **Libraries:** `pandas`, `numpy`, `yfinance`, `plotly`, `jinja2`, `pdfkit`
* **Environment:** Google Colab / Jupyter Notebook
